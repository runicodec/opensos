# OpenSoS (Open Spirits of Steel)

OpenSoS is a grand strategy war game where you guide a country through diplomacy, industry, politics, military buildup, and war.

## How To Launch

No end-user releases are available yet. Contributors can build from source on Windows, macOS, and Linux — see [BUILDING.md](BUILDING.md) for instructions.

- **Windows**: CMake + MSVC + vcpkg
- **macOS / Linux**: devenv (Nix)

## Basic Controls

> Note: This section needs to be rewritten as a lot of the controls listed are incorrect.

- `WASD` or arrow keys: move the camera
- Mouse wheel or `[` `]`: zoom the map
- `Space`: open or close the sidebar
- `Esc`: open the pause menu or dismiss popups
- Left click: select provinces, countries, or units
- Left click and drag: box-select multiple divisions
- Right click: move selected divisions
